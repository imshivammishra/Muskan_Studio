# Muskan Studio — Portfolio Website

A modern, minimal, responsive portfolio website for Muskan Studio — a creative design studio.

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript (ES6)

## Features

- Fully responsive across all devices
- Sticky navigation with blur effect on scroll
- Animated statistics counter
- Portfolio filter by category
- Contact form with client-side validation
- Intersection Observer for scroll animations
- Semantic HTML with ARIA labels
- SEO meta tags and structured data
- Print and reduced-motion media queries

## Setup

1. Open `index.html` in a browser
2. Replace placeholder images in `assets/images/` with actual images
3. Update social links and contact info in `index.html`
4. Deploy to your hosting provider
